FITCLUB responsive final

Структура:
- index.html
- about.html
- trainers.html
- contacts.html
- css/style.css
- js/main.js
- images/*.webp
- favicon.svg

Виконано під вимоги лабораторної №5:
- додано viewport;
- адаптивна верстка для смартфона, планшета і ПК;
- немає горизонтальної прокрутки;
- меню доступне з кожної сторінки;
- телефон активний через tel:;
- форма коротка та має autocomplete;
- шрифт читабельний, активні елементи достатнього розміру;
- зображення оптимізовані у WebP.
